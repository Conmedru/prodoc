# Prodoctorov.ru Doctor Scraper

Скрапер для сбора данных врачей с сайта prodoctorov.ru.

## Быстрый старт (бесплатный запуск в облаке)

### Через GitHub Actions (рекомендуется)

1. **Создай репозиторий на GitHub** и загрузи эти файлы
2. **Запусти workflow вручную:**
   - Зайди в раздел **Actions** → **Scrape Prodoctorov Doctors**
   - Нажми **Run workflow**
   - Выбери `sample_only: true` для теста (20 врачей) или `false` для полного сбора
3. **Скачай результат** — через ~2-6 минут (sample) или ~2-6 часов (full) артефакты будут доступны для скачивания

### Лимиты GitHub Actions
- **Бесплатно:** 2000 минут/месяц (для публичных репо — безлимитно)
- **Таймаут одного запуска:** 6 часов (достаточно для ~5000 врачей)
- **Хранение артефактов:** 30 дней

## Локальный запуск

```bash
# Установка зависимостей
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Тестовая выборка (20 врачей, ~2 минуты)
python sample_20.py

# Полный пилот (4 специальности, Москва, ~2-6 часов)
python main.py
```

## Результаты

Excel файл создается в папке `output/`:
- `sample_20_doctors.xlsx` — тестовая выборка
- `doctors_YYYYMMDD_HHMMSS.xlsx` — полный сбор

### Структура Excel

| Колонка | Описание |
|---------|----------|
| № | Порядковый номер |
| ФИО | Полное имя врача |
| Город | Город |
| Специальность | Все специальности через запятую |
| Место работы | Текущие организации (м.б. несколько) |
| Адрес работы | Адреса для текущих организаций |
| Стаж | Опыт работы (лет/годов) |
| URL | Ссылка на карточку врача |

## Текущий пилот

- **Город:** Москва
- **Специальности:** Офтальмологи, Детские офтальмологи, Педиатры, Неврологи
